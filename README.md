# Budget Tracker 


## Purpose
Generate a budget tracker with offline capabilities. When offline, application will still save entries and later post when internet is reconnected.

## Built With
* HTML
* CSS
* Javascript
* Express.js
* Node
* MongoD
* Mongoose
